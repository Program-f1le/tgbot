Инструкция к запуску телеграмм – бота @ScheduleForSiriusStudents_bot
Установить на устройства Telegram:
Для веб-версии. Откройте любой браузер (например, Chrome, Opera, Firefox, Safari) и перейдите на web.telegram.org. После авторизации добавьте сайт в закладки, чтобы каждый раз не искать адрес заново. 
Для отдельной установки приложения. Скачайте Telegram Desktop с официального сайта (desktop.telegram.org).
Выберите версию программы для своей операционной системы (Windows, macOS, Linux) и нажмите на кнопку загрузки. После завершения загрузки откройте установочный файл и следуйте инструкциям мастера установки: 
Выберите язык установки. 
Укажите папку, куда будет установлено приложение (по умолчанию это «Program Files» для Windows). 
Определите, хотите ли вы создать ярлык на рабочем столе и добавить программу в меню «Пуск». 
Нажмите «Установить» и дождитесь завершения процесса. Установка Telegram Desktop займёт несколько минут, после чего программа автоматически запустится. 
Ещё Telegram можно скачать в Microsoft Store. Найдите Microsoft Store в меню «Пуск», в строке поиска введите «Telegram» и откройте его.
Кликните по кнопке «Получить». Скачивание бесплатное, но нужно авторизоваться в учётной записи Microsoft. 
После успешной авторизации на рабочем столе появится иконка Telegram Desktop. 
Для устройств на базе Android: 
Откройте Google Play. 
Введите в строке поиска «telegram». 
Нажмите на кнопку «установить». 
После установки откройте приложение.
Введите свои данные. 
Дождитесь смс с кодом подтверждения, введите его в открывшемся поле. 
Укажите свои имя и фамилию, подтвердите регистрацию.
Для устройств на базе iPhone или iPad:  
Откройте AppStore.  
Через поле поиска найдите мессенджер.  
Нажмите на кнопку установки.  
Когда Telegram будет инсталлирован на устройство, откройте его. 
Выберите страну и введите номер телефона. 
Подтвердите регистрацию кодом из смс. 
Введите свои данные. 
Ещё один способ — установить Telegram с официального сайта:  
Зайдите на официальный сайт мессенджера — telegram.org. 
На главной странице выберите свою операционную систему: Android, iOS, Windows, Linux, macOS.  
После скачивания приложения найдите его в загрузках и запустите. 
Возможно, потребуется разрешение на установку — если устройство определит telegram.org как «неизвестный источник».

3. Функционал бота:
Доступные команды:
/today - расписание на сегодня
/tomorrow - расписание на завтра
/week - расписание на неделю
/add - добавить событие
/help - список команд
/schedule - инструкция по добавлению событий
Напоминание о предстоящем личном событии за 1 час и за 15 минут до события.
