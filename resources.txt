В начале работы с ботом обязательно нужно использовать /start !!!

Для работы программы нужны:

sql база данных(желательно postgresql)

python 3.12.0 и выше (Если на более новых версиях программа работает некорректно - установите python 3.12.0).

Для проверки установлен ли python и какая у него версия используйте в командной строке или в терминале:
python

Библиотеки python:
anyio 4.3.0
SQLAlchemy 2.0.30
aiogram 3.21.0
datetime - стандартная библиотека

Для установки библиотеки введите в командной строке или в терминале:
pip install название библиотеки

Для проверки установлена ли библиотека и какая у нее версия используйте в командной строке или терминале:
pip list

Чтобы бот работал нужно перенести все файлы проекта в одну папку, запустить файл main.py в командной строке или терминале.
Бот выполняет свои функции ПОКА файл main.py запущен.
Файл main можно запустить на удаленном сервере(который можно арендовать).
Или использовать свой пк в качестве сервера.
Для запуска файла Python из командной строки необходимо открыть терминал (в Windows это командная строка или PowerShell, в Linux/macOS - терминал) и ввести команду python <путь_к_файлу.py> или python3 <путь_к_файлу.py>, в зависимости от установленной версии Python. Путь к файлу может быть абсолютным или относительным.
Подробная инструкция:

1. Откройте терминал:
Windows: Нажмите Win + R, введите cmd и нажмите Enter. Или найдите "Командная строка" в меню Пуск.
macOS: Нажмите Command + Space, введите terminal и нажмите Enter.
Linux: Нажмите Ctrl + Alt + T, или найдите "Терминал" в меню приложений.

2. Перейдите в директорию, где находится файл:
Если файл не в текущей директории, используйте команду cd <путь_к_директории> для навигации. Например, cd Documents/python_scripts.

3. Запустите файл:
Введите команду python <имя_файла.py> или python3 <имя_файла.py>, где <имя_файла.py> - это имя вашего Python файла.
Если в системе установлены обе версии Python (2 и 3), рекомендуется использовать python3, чтобы избежать путаницы.
Если вы работаете с интерактивной оболочкой Python, введите python или python3 и нажмите Enter. Затем введите код и нажмите Enter, чтобы выполнить его.

Чтобы остановить выполнение файла Python из командной строки, можно использовать сочетание клавиш Ctrl+C.
Это отправит сигнал SIGINT (сигнал прерывания) интерпретатору Python, который прервет выполнение программы.
Если это не сработает, можно использовать команду taskkill в Windows для принудительного завершения процесса Python, например, taskkill /im python.exe /f /t.

Другие способы:
В некоторых случаях, если Python запущен в интерактивном режиме, можно ввести exit() или quit() для выхода из интерпретатора.
Также, в некоторых терминалах, Ctrl+Z может использоваться для приостановки процесса, а не для его завершения.
